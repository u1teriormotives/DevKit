{DOCUMENT /}
{document /}                        -> declares document type (i.e., what it was compiled from, if it was handwritten or not, language, etc.)
    self-contained
    Attributes: (t="type")      => what it was compiled from or if it was handwritten (e.g., "html5", "java", "js_ts", "csharp", "c", "dk", etc.; random metadata)
                (l="language")  => the language the source content is in (RFC 5646)
[document $:tree]   -> encompassing document tree
[head $:head]       -> encompassing metadata tree (head)
[body $:body]       -> encompassing viewable data tree (body)
[meta $:randomId]                   -> metadata element
    uses closing tag
    Attributes: (name="key" /)          => the key
                (content="value" /)     => the value
                (http-equiv="key" /)    => modifies header
[title $:title]                     -> webpage title
    uses closing tag
    uses text content
[external-source $:randomId]        -> marks an external source (e.g., stylesheet, script)
    uses closing tag
    Attributes: (type="type" /)         => the type of source ("stylesheet", "script")
                (source-url="url" /)    => the url of the source (must be valid uri; validated at compile-time)
[internal-source $:randomId]        -> marks an internal source (e.g., stylesheet, script)
    uses closing tag
    uses text content
    Attributes: (type="type" /)         => the type of source ("stylesheet", "script")
                (content-type="type" /) => the content-type of the source (MIME Type)
[element::namespace $:randomId]     -> element that modifies the viewable content
    uses closing tag
    uses text content (optional)
    uses shadow text content (optional; i.e., default text if it is a writeable element)
    can have children
    Attributes: (active="boolean" /)                    => if the element is viewable
                (alt-content="value" /)                 => alternative text for an embedded, visual source; fallback description
                                                       if request for source fails
                (anchor="value" /)                      => named anchor for deep linking within a document
                (autocomplete="boolean" /)              => whether the client may request completion
                (autoplay="boolean" /)                  => for media elements; if playback is immediate
                (bind-direction="value" /)              => data flow direction ("one-way", "two-way")
                (bind-property="value" /)               => specific property on bind-ref element being observed
                (bind-ref=$:id /)                       => binds this element's state to another's
                (caption-url="url" /)                   => caption/subtitle source url (validated at compile time; accepts only text/tsv)
                (checked="boolean" /)                   => if a "checkbox" is toggled
                (content-policy="value" /)              => csp scoped to the element
                (content-type="type" /)                 => MIME type of the embedded source
                (control-ref=$:id /)                    => declares this element is the master to another
                (controls="boolean" /)                  => if controls are exposed for a media element
                (cross-origin="value" /)                => cross-origin handling ("anonymous", "use-credentials")
                (download="boolean" /)                  => if target-url should be downloaded instead of displayed
                (download-filename="value" /)           => the name of the downloaded file if (download="true" /)
                (draggable="boolean" /)                 => if the element is able to be dragged by the client
                (drop-target="boolean" /)               => if the element can receive dragged elements
                (emit-on="value" /)                     => condition on which the event is fired ("change", "blur", "load", etc.)
                (event-handler={event, $:id} /)         => reference to a handler for an event
                (event-trigger=[event] /)               => declares what events the element emits
                (expanded="boolean" /)                  => if an collapsible is expanded or not
                (focusable="boolean" /)                 => controls if the element can be tabbed onto
                (group-ref="value" /)                   => logical group name this element belongs to
                (hidden="boolean" /)                    => alias for active
                (indeterminate="boolean" /)             => if an element is neither selected nor not selected (or checked and vice-versa)
                (input-pattern="regex" /)               => validated pattern of user input for writeable content
                (input-type="type" /)                   => expected type of input for a writeable element
                (interactable="boolean" /)              => controls if events like "hover", "click", etc. are applicable for the element
                (lazy-load="boolean" /)                 => defers loading until element enters viewport
                (lazy-render="boolean" /)               => defers rendering until element is near viewport
                (loop="boolean" /)                      => for media elements; whether playback loops
                (max-length="integer" /)                => the maximum length of text content in an element
                (max-select="integer" /)                => the maximum amount of selected options capable
                (max-value="integer" /)                 => maximum integer for a writeable element that accepts numbers
                (min-length="integer" /)                => minimum length of text content in a writeable element
                (min-select="integer" /)                => the minimum amount of selected options capable
                (min-value="integer" /)                 => minimum integer for a writeable element that accepts numbers
                (multiline="boolean" /)                 => whether the client may insert a newline in their response
                (multi-select="boolean" /)              => if multiple child options can be selected or not
                (muted="boolean" /)                     => for media elements; if audio is audible
                (no-referrer="boolean" /)               => suppresses referrer information when following target-url
                (once="boolean" /)                      => if the element does not update after first render
                (open-behavior="value" /)               => how to open target ("self", "new", "overlay")
                (placeholder="value" /)                 => default text content; alias to shadow text content
                (playback-end="number" /)               => the point at which playback ends in media
                (playback-start="number" /)             => the point at which playback starts in media
                (position={x, y, z} /)                  => keyset for position (auto-calculated or set by user)
                (poster-url="url" /)                    => preview image shown before media playback (validated at compile time)
                (pressed="boolean" /)                   => if a button element is depressed
                (read-only="boolean" /)                 => if a writeable element cannot be written to
                (rel="value" /)                         => relationship between this element and target-url
                (render-priority="integer" /)           => priority hint (lowest -> first, highest -> last)
                (required="boolean" /)                  => if a writeable element is required to have something in it
                (sandbox-polity="value" /)              => what the element may do
                (sandboxed="boolean" /)                 => if the element is separated from the rest of the document
                (selectable="boolean" /)                => if element text can be selected or not
                (selected="boolean" /)                  => if an element is currently selected by the client in a group of options
                (set-classification="value" /)          => user-set class (i.e., html's "class" attribute)
                (set-data={"key", "value"} /)           => user-defined keypair
                (set-identifier="value" /)              => user-set id (i.e., html's "id" attribute)
                (slot="value" /)                        => named insertion point within a parent's structure
                (sort-index="integer" /)                => explicit sort order within a parent or group
                (source-url="value" /)                  => url of an embedded source (e.g., image; validated at compile time)
                (spellcheck="boolean" /)                => whether the client may check spelling
                (step-value="integer" /)                => increment for numerical input
                (tab-index="integer" /)                 => explicit position in focus order
                (target-ref=$:id /)                     => internal reference to another element by internal identifier
                (target-url="url" /)                    => destination url (validated at compile-time)
                (trim-whitespace="boolean" /)           => whether leading and trailing whitespace is automatically trimmed
                (user-agent-classification="value" /)   => key/value pair for the compiled source (what would html5 call it)
                (volume="float" /)                      => initial volume level (0.0-1.0)
                (writeable="boolean" /)                 => if the client can write in it (e.g., textbox)
    Namespaces: paragraph               => a default paragraph text element
                header                  => a header text element, depends on (user-agent-classification /) & (set-classification /) for its size
                link                    => a followable link
                label                   => a visible label bound to another element (e.g., a caption for a figure)
                search                  => a search region
                vertical_separator      => a vertical line
                horizontal_separator    => a horizontal line
                line_break              => a linebreak
                media::subspace         => a media element, depends on a subspace for its classification
                    media::visual   ==> a visual element (e.g., image, video)
                    media::audio    ==> an audio element (e.g., mp3 file)
                structure::subspace     => an element for structure purposes, depends on a subspace for its classification
                    structure::divider          ==> a content division element (i.e., the div element in html)
                    structure::span             ==> a content span element (i.e., the span element in html)
                    structure::code             ==> a codeblock
                    structure::quote            ==> a quote
                    structure::citation         ==> a citation
                    structure::aside            ==> an aside
                    structure::caption          ==> a caption
                    structure::details          ==> a collapsible details section
                    structure::dialog           ==> dialog
                    structure::menu             ==> a menu
                    structure::main             ==> the main content
                    structure::section          ==> a section of the content
                    structure::subsection       ==> a subsection of a section
                    structure::navigator        ==> a navigation bar
                    structure::footer           ==> trailing content
                    structure::figure           ==> a media/content unit
                    structure::banner           ==> content block similar to a header
                    structure::header           ==> leading content
                    structure::tooltip          ==> contextual helper content
                    structure::warning          ==> warning callout
                    structure::notice           ==> informational callout
                text::subspace          => inline or semantic text elements, depends on a subspace for its classification
                    text::abbreviation  ==> abbreviation with expansion
                    text::definition    ==> defining instance of a term
                    text::keyboard      ==> keyboard input text
                    text::sample        ==> sample output
                    text::variable      ==> variable name
                    text::time          ==> timestamp or date
                form::subspace          => interactive input and form elements, depends on a subspace for its classification
                    form::form          ==> container for input controls
                    form::input         ==> generic user input field
                    form::textbox       ==> textual input
                    form::textarea      ==> multiline text input
                    form::checkbox      ==> boolean checkbox
                    form::radio         ==> single-selection radio option
                    form::button        ==> interactive button
                    form::submit        ==> form submission control
                    form::label         ==> label for input element
                    form::select        ==> selectable list container
                    form::option        ==> selectable option
                    form::option_group  ==> group of options
                    form::progress      ==> progress indicator
                    form::meter         ==> bounded numeric gauge
                    form::fieldset      ==> grouping container
                    form::legend        ==> label for fieldset
                table::subspace         => tabular data structures, depends on a subspace for its classification
                    table::table        ==> table container
                    table::header       ==> header row group
                    table::body         ==> table body
                    table::footer       ==> table footer
                    table::row          ==> table row
                    table::cell         ==> table data cell
                    table::header_cell  ==> header cell
                    table::column_group ==> column grouping
                    table::column       ==> column definition
                article::subspace       => content-blocks, depends on a subspace for its classification
                    article::article    ==> standalone article/document entry
                    article::post       ==> post-like entry
                    article::entry      ==> generic feed/article entry
                    article::commentary ==> comment/reply block
                    article::summary    ==> summary/abstract block
                    article::address    ==> contact/authorship block
                list::subspace          => lists, depends on a subspace for its classification
                    list::ordered       ==> ordered list
                    list::unordered     ==> unordered list
                    list::item          ==> list item
                    list::description   ==> description list
                    list::term          ==> description term
                    list::detail        ==> description detail
